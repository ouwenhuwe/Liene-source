热门问题

如何连接小米打印机到Wi-Fi网络？

+

1. 确保打印机已开机并处于待机状态。
2. 打开小米打印APP，点击"添加打印机"。
3. 按照APP提示操作，选择您的Wi-Fi网络并输入密码。
4. 等待连接成功提示。

打印机显示"缺纸"但纸盒中有纸怎么办？

+

1. 检查纸张是否放置正确，确保纸张平整无褶皱。
2. 检查纸张传感器是否有异物阻挡。
3. 重启打印机尝试。
4. 如果问题持续，请联系客服。

如何更换打印机的墨盒？

+

1. 打开打印机前盖。
2. 等待打印头移动到更换位置。
3. 按下墨盒卡扣，取出旧墨盒。
4. 拆开新墨盒包装，安装到打印机中。
5. 关闭前盖，完成更换。

手机无法发现打印机怎么办？

+

1. 确保手机和打印机连接在同一Wi-Fi网络下。
2. 检查打印机是否处于可被发现状态。
3. 重启手机和打印机。
4. 更新小米打印APP到最新版本。
